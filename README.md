# Next-Mart - eCommerce App (Next.js + DummyJSON API)

Next-Mart is a simple eCommerce web application built with **Next.js** that displays products using the **DummyJSON API**.  
It includes product listing, details pages, and cart functionality powered by **Zustand** for state management.

---

## Live Preview
https://arnextmart.vercel.app/

---

## Features
- Browse products fetched from DummyJSON API  
- View product details  
- Add and remove items from the cart  
- Manage cart state using Zustand  
- Responsive UI built with Tailwind CSS  
- Deployed on Vercel  
- Fast client-side navigation with Next.js App Router  

---

## Tech Stack
| Layer | Technology |
|--------|-------------|
| Framework | Next.js 14 (App Router) |
| Styling | Tailwind CSS |
| API | DummyJSON (https://dummyjson.com) |
| State Management | Zustand |
| Deployment | Vercel |
